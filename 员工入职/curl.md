# curl

- 默认是 GET 方法
- curl 下载文件的时候默认下载到当前的目录

| 参数 | 作用                                                |
| ---- | --------------------------------------------------- |
| -O   | 下载文件并重命名，默认使用 URL 的最后部分当作文件名 |
| -v   | 显示通信过程                                        |
|      |                                                     |
